In questa sezione è possibile visualizzare in dettaglio i dati sulle scuole di provenienza degli studenti della Bicocca. Nella parte a sinistra della tabella è possibile effettuare query alla tabella stessa, utilizzando tutte le variabili disponibili. Le query vengono effettuate in “AND” sulle tutte le righe della tabella.

Dare sempre il tempo al sistema di elaborare le informazioni, vista la grande quantità di dati da trattare.
